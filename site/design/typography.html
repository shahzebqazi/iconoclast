Typography

Design reference — font stack and type scale.

Font stack

The site uses the system UI stack for fast loading and native feel. Monospace is reserved for code and token names.

body

The quick brown fox jumps over the lazy dog. 0123456789 — em dash, “smart quotes”, and inline code when needed.

font-family (from site/style.css)

system-ui, -apple-system, Segoe UI, Roboto, Ubuntu, Cantarell, Noto Sans, sans-serif

Monospace (code, hints)

ui-monospace, SFMono-Regular, Menlo, Monaco, Consolas, monospace

Type scale

Approximate sizes used on the public pages and design docs. The live site uses line-height: 1.55 on :root in site/style.css.

Role Size Weight
Site title (h1) clamp(1.5rem, 4vw, 1.85rem) 700
Section label (h2) 0.8rem, uppercase 600
